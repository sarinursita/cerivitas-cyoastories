# Cerivitas CYOA Stories 🌳📖

Choose Your Own Adventure stories by Agent Adventure Workshop participants!

## 📁 Structure

Each participant gets their own folder:

```
rina-cookies/
  README.md        ← author info
  1.md             ← scene 1
  1.1.md           ← scene 1.1
  1.2.md           ← scene 1.2
  1.1.1.md         ← scene 1.1.1
  ...
  images/          ← optional images
```

## 🌐 GitHub Pages

Every folder = a playable CYOA at:
`https://sarinursita.github.io/cerivitas-cyoastories/[folder-name]/`

## 📝 How to Write

1. **Plan your tree** (outline with numbers)
2. **Write each scene** in markdown
3. **Link choices** to other scene files
4. **Add plot twist** in one branch!
