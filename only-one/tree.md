# Branching Tree — Only One

```
Scene 1: Hari Biasa yang Nggak Biasa
  ├── Nyapa Hali duluan → Scene 2A
  └── Nyapa Kaizo duluan → Scene 2B

Scene 2A: Ngobrol Bentar Sama Hali
  ├── Ajak Hali ngopi → Scene 3A
  └── Balik ke kelas bareng → Scene 3B

Scene 2B: Ketemu Kaizo di Koridor
  ├── Bahas rapat OSIS → Scene 3C
  └── Diam aja, salting → Scene 3D

Scene 3A: Ngopi di Kantin
  ├── Tanya balik serius → Scene 4A
  └── Bercanda biar cair → Scene 4A

Scene 3B: Balik ke Kelas Bareng
  ├── Kejar dan tanya → Scene 4A
  └── Diem aja sambil senyum → Scene 4A

Scene 3C: Rapat OSIS
  ├── Nanya langsung → Scene 4B
  └── Balas santai → Scene 4B

Scene 3D: Momen Hening
  ├── Panggil balik Kaizo → Scene 4B
  └── Senyum dan simpan cokelat → Scene 4B

Scene 4A: Semakin Dekat dengan Hali
  ├── Nunggu sampe selesai → Scene 5A
  └── Pulang duluan → Scene 5A

Scene 4B: Semakin Dekat dengan Kaizo
  ├── Nanya balik → Scene 5B
  └── Godain balik → Scene 5B

Scene 5A: Konflik (Hali)
  ├── Memaafkan Hali → Scene 6A
  └── Masih kesel → Scene 6A

Scene 5B: Konflik (Kaizo)
  ├── Ngejar Hali duluan → Scene 6B
  └── Tetap sama Kaizo → Scene 6B

Scene 6A: Penjelasan (Hali)
  ├── Peluk Hali → Scene 7A (Ending Hali) ❤️
  └── Bilang jujur → Scene 7A (Ending Hali) ❤️

Scene 6B: Setelah Pengakuan (Kaizo)
  ├── Terima → Scene 7B (Ending Kaizo) 🧡
  └── Minta waktu → Scene 7B (Ending Kaizo) 🧡

Scene 7A: Hanya Satu — Ending Hali 🏁
Scene 7B: Hanya Satu — Ending Kaizo 🏁
```
