# Branching Tree — Only One

```
Scene 1: Intro Aysel
  └── Pilih: Nyapa Hali (Scene 2A) atau Nyapa Kaizo (Scene 2B)

Scene 2A: Ngobrol sama Hali
  └── Pilih: Ajak ngopi (Scene 3A) atau Balik kelas bareng (Scene 3B)

Scene 2B: Ketemu Kaizo
  └── Pilih: Bahas rapat OSIS (Scene 3C) atau Momen hening (Scene 3D)

Scene 3A/3B/3C/3D → Scene 4A (Hali path) atau Scene 4B (Kaizo path)
  → Scene 5A/5B (Konflik)
  → Scene 6A/6B (Resolusi)
  → Scene 7: ENDING (Hali ❤️ atau Kaizo ❤️)
```
