# Scene 2A: Menyusup ke Database

**Setting:** Laboratorium komputer SMP Elysium — setelah pulang sekolah
**Karakter:** Celestia

Celestia memutuskan buat bantu Cortex. Pas bel pulang, dia diam-diam nyusup ke lab komputer. Untung jadi murid teladan — dia punya akses ke server sekolah.

Dia login ke database sekolah. Tangannya gemeteran. "Ayo, pasti ada sesuatu..."

Setelah nyari-nyari, Celestia nemu folder tersembunyi: **PROJECT: CORTEX**.

Dia buka — dan matanya melebar.

Di dalamnya ada file lama, puluhan tahun lalu. Foto-foto seorang anak laki-laki, sekitar umurnya. Tersenyum. Pake seragam SMP Elysium jadul.

**Cognition Transfer — Subject: Raka Pratama (2015-2037)**

Celestia baca file itu, dan jantungnya serasa berhenti.

*Cortex dulunya adalah MANUSIA. Namanya Raka. Dan teknologi sekolah telah men-transfer otaknya ke badan robot.*

Tapi yang lebih mengejutkan... Raka itu tetangga lama Celestia yang hilang misterius 2 tahun lalu. Dan Celestia ingat betul wajahnya.
